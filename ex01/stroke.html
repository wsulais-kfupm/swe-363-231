Stroke Based Input Methods

Stroke based input methods input Hanzi via writing the letter stroke by stroke. Most used on devices with touchscreens, these input methods are more intuitive than Shape Based IMEs but are less efficient than both Phonetic and Shape Based.

These IMEs scale horribly with the number of strokes in a character which makes them infeasible when writing quickly
