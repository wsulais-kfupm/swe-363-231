Shape Based Input Methods

Shape based input methods input Hanzi by assigning each character to a code, usu. ASCII letters. It is often desirable to have a consistent logical mapping for the codes which often entails assigning the codes via the character's radicals.

What are radicals?

A radical (部首) is a graphical component of a Hanzi character. These components help classify characters in dictionaries

These components sometimes give semantics (meaning) or phonetics (reading/pronunication) to the character. For the above example, the radical 馬 gives the phonetic component mā while the radical 女 gives the meaning of woman, mother.
