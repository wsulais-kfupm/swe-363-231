CJK Input Methods

This site contains information about the various types of input methods available for CJK languages

Han Unification

Due to the breadth of the CJK world, regional character variants are to be expected. Furthermore, simplifications and/or writing reformations such as the Chinese Simplification or Japanese script reform have caused the characters to diverge.

The unicode consortium faced the issue of trying to map these different variants especially when merging existing encoding schemes.

In Web

HTML provides utilities to specify the language and region of the text via the lang global attribute. For example, 

  <span lang="zh-CN"> 返 </span>

would display the character「返」how it would be in Mainland China.
